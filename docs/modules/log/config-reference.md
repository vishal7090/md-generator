# Log config reference

| Section | Purpose |
| --- | --- |
| `incidents.min_occurrences` | Minimum repeats to open an incident |
| `chunking.strategies` | Semantic chunk strategies to emit |
| `embeddings.exporters` | `jsonl`, `parquet`, `chroma`, `faiss` |
| `correlation.enabled` | Request/trace correlation artifacts |
| `knowledge_graph.enabled` | Service dependency graph export |
| `timeline.enabled` | Reconstructed operational timeline |
| `intelligence.enabled` | Heuristic root-cause summaries |
